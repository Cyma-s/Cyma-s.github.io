IoC ContainerлһҖ?

к°Ғк°Ғмқҳ мҡ©м–ҙлҘј н’Җм–ҙліҙмһҗ.

IoC лһҖ Inversion of Control мқҳ м•ҪмһҗлЎңм„ң, м ңм–ҙмқҳ м—ӯм „мқ„ лң»н•ңлӢӨ. мқҳмЎҙм„ұ мЈјмһ…мқҙлқјкі  н•ҳкё°лҸ„ н•ңлӢӨ. м»Ён…Ңмқҙл„ҲлҠ” к°қмІҙмқҳ мғқлӘ…мЈјкё°лҘј кҙҖлҰ¬н•ҳкі , мғқм„ұлҗң мқёмҠӨн„ҙмҠӨл“Өм—җкІҢ м¶”к°Җм Ғмқё кё°лҠҘмқ„ м ңкіөн•ңлӢӨ.

мҰү, IoC ContainerлҠ” к°қмІҙмқҳ мғқм„ұкіј лқјмқҙн”„мӮ¬мқҙнҒҙмқ„ кҙҖлҰ¬н•ҳкі , мқҳмЎҙм„ұмқ„ нҒҙлһҳмҠӨм—җ мЈјмһ…н•ҳкё°лҸ„ н•ңлӢӨ.

мЈјлЎң лӢӨмқҢкіј к°ҷмқҖ кё°лҠҘмқ„ м ңкіөн•ңлӢӨ.

  • IoC м»Ён…Ңмқҙл„ҲлҠ” к°қмІҙмқҳ мғқм„ұмқ„ мұ…мһ„м§Җкі , мқҳмЎҙм„ұмқ„ кҙҖлҰ¬н•ңлӢӨ.
  • POJOмқҳ мғқм„ұ, мҙҲкё°нҷ”, м„ңл№„мҠӨ, мҶҢл©ём—җ лҢҖн•ң к¶Ңн•ңмқ„ к°Җм§„лӢӨ.
  • к°ңл°ңмһҗл“Өмқҙ м§Ғм ‘ POJOлҘј мғқм„ұн•  мҲҳ мһҲм§Җл§Ң м»Ён…Ңмқҙл„Ҳм—җкІҢ л§ЎкёҙлӢӨ.
  • к°ңл°ңмһҗлҠ”В л№„мҰҲлӢҲмҠӨ лЎңм§Ғм—җ м§‘мӨ‘н•  мҲҳ мһҲлӢӨ.
  • к°қмІҙ мғқм„ұ мҪ”л“ңк°Җ м—ҶмңјлҜҖлЎң TDDк°Җ мҡ©мқҙн•ҳлӢӨ.

Spring Frameworkмқҳ IoC м»Ён…Ңмқҙл„Ҳ кө¬нҳ„мқҖ org.springframework.beans, org.springframework.context нҢЁнӮӨм§Җмқҳ кё°л°ҳмқҙ лҗңлӢӨ.

к·ё мӨ‘м—җм„ңлҸ„ BeanFactory мқён„°нҺҳмқҙмҠӨлҠ” лӘЁл“  мң нҳ•мқҳ к°қмІҙлҘј кҙҖлҰ¬н•  мҲҳ мһҲлҠ” кі кёү Configuration л©”м»ӨлӢҲмҰҳмқ„ м ңкіөн•ңлӢӨ. BeanFactory мҷҖ ApplicationContext мқён„°нҺҳмқҙмҠӨлҠ” Spring IoC м»Ён…Ңмқҙл„ҲлҘј к°ҖлҰ¬нӮЁлӢӨ.

Spring IoC м»Ён…Ңмқҙл„Ҳк°Җ кҙҖлҰ¬н•ҳлҠ” к°қмІҙлҘјВ л№Ҳ(bean) мқҙлқјкі  н•ҳкі , мқҙ л№Ҳ(bean)л“Өмқ„ кҙҖлҰ¬н•ңлӢӨлҠ” мқҳлҜёлЎң м»Ён…Ңмқҙл„ҲлҘј л№Ҳ нҢ©нҶ лҰ¬(BeanFactory)В лқјкі  л¶ҖлҘёлӢӨ.

  • к°қмІҙмқҳ мғқм„ұкіј к°қмІҙ мӮ¬мқҙмқҳ лҹ°нғҖмһ„(run-time) кҙҖкі„лҘј DI кҙҖм җм—җм„ң ліј л•ҢлҠ” м»Ён…Ңмқҙл„ҲлҘјВ BeanFactoryлқјкі  н•ңлӢӨ.
  • BeanFactoryм—җ м—¬лҹ¬ к°Җм§Җ м»Ён…Ңмқҙл„Ҳ кё°лҠҘмқ„ м¶”к°Җн•ҳм—¬В м• н”ҢлҰ¬мјҖмқҙм…ҳ м»Ён…ҚмҠӨнҠё(ApplicationContext) лқјкі  л¶ҖлҘёлӢӨ.

BeanFactoryмҷҖ ApplicationContext

  • BeanFactory Bean FactoryлҠ” Beanмқ„ л“ұлЎқ, мғқм„ұ, мЎ°нҡҢ, л°ҳнҷҳмқ„ кҙҖлҰ¬н•ңлӢӨ. ліҙнҶөмқҖ BeanFactoryлҘј л°”лЎң мӮ¬мҡ©н•ҳм§Җ м•Ҡкі , мқҙлҘј нҷ•мһҘн•ң ApplicationContextлҘј мӮ¬мҡ©н•ңлӢӨ. getBean() л©”м„ңл“ңк°Җ м •мқҳлҗҳм–ҙ мһҲлӢӨ.

  • ApplicationContext Beanмқ„ л“ұлЎқ,мғқм„ұ,мЎ°нҡҢ,л°ҳнҷҳмқ„ кҙҖлҰ¬н•ҳлҠ” кё°лҠҘмқҖ BeanFactoryмҷҖ к°ҷлӢӨ. Springмқҳ к°Ғмў… л¶Җк°Җ м„ңл№„мҠӨлҘј м¶”к°ҖлЎң м ңкіөн•ңлӢӨ.

ApplicationContext

м–ём ң мӢӨн–үлҗҳлӮҳ?

@SpringBootApplication  
public class JwpCartApplication {  
    public static void main(String[] args) {  
        SpringApplication.run(JwpCartApplication.class, args);  
    }   
}

мһҘл°”кө¬лӢҲ лҜём…ҳмқҳ SpringApplication.run мқ„ мӢӨн–үн•ҙліҙмһҗ.

SpringApplication мқҳ run мқҙ мӢӨн–үлҗңлӢӨ.

кі„мҶҚ нғҖкі  л“Өм–ҙк°Җл©ҙ SpringApplication мқҳ run л©”м„ңл“ңк°Җ мӢӨн–үлҗңлӢӨ.

public ConfigurableApplicationContext run(String... args) {  


   long startTime = System.nanoTime();


// BootStrapContext мғқм„ұ
   DefaultBootstrapContext bootstrapContext = createBootstrapContext();  
   ConfigurableApplicationContext context = null;  


// Java AWT Headless Property м„Өм •
   configureHeadlessProperty();  


// мҠӨн”„л§Ғ м• н”ҢлҰ¬мјҖмқҙм…ҳ лҰ¬мҠӨл„Ҳ мЎ°нҡҢ л°Ҹ starting мІҳлҰ¬
   SpringApplicationRunListeners listeners = getRunListeners(args);  
   listeners.starting(bootstrapContext, this.mainApplicationClass);  
   
   try {  


    // Arguments лһҳн•‘ л°Ҹ Environment мӨҖл№„
      ApplicationArguments applicationArguments = new DefaultApplicationArguments(args);  
      ConfigurableEnvironment environment = prepareEnvironment(listeners, bootstrapContext, applicationArguments);  


    // IgnoreBeanInfo м„Өм •
      configureIgnoreBeanInfo(environment);  


    // л°°л„Ҳ м¶ңл Ҙ
      Banner printedBanner = printBanner(environment);  


    // **м• н”ҢлҰ¬мјҖмқҙм…ҳ м»Ён…ҚмҠӨнҠё мғқм„ұ**
      context = createApplicationContext();  
      context.setApplicationStartup(this.applicationStartup);  


    // **Context мӨҖл№„ лӢЁкі„**
      prepareContext(bootstrapContext, context, environment, listeners, applicationArguments, printedBanner);  


    // Context Refresh лӢЁкі„
      refreshContext(context);  


    // Context Refresh нӣ„мІҳлҰ¬ лӢЁкі„
      afterRefresh(context, applicationArguments);  


    // мӢӨн–ү мӢңк°„ м¶ңл Ҙ л°Ҹ лҰ¬мҠӨл„Ҳ started мІҳлҰ¬
      Duration timeTakenToStartup = Duration.ofNanos(System.nanoTime() - startTime);  
      if (this.logStartupInfo) {  
         new StartupInfoLogger(this.mainApplicationClass).logStarted(getApplicationLog(), timeTakenToStartup);  
      }  
      listeners.started(context, timeTakenToStartup);  


    // Runners мӢӨн–ү
      callRunners(context, applicationArguments);  
   }  
   catch (Throwable ex) {  
      handleRunFailure(context, ex, listeners);  
      throw new IllegalStateException(ex);  
   }  
   ...
   return context;  
}

кі§ м§Җмҡё лӮҙмҡ©

BootStrapContext мғқм„ұ

createBootstrapContext() лӮҙл¶ҖлЎң л“Өм–ҙк°Җліҙмһҗ.

createBootstrapContext()лҠ” ApplicationContextлҘј мғқм„ұн•ҳкё° м „м—җ BootstrapContextлҘј мғқм„ұн•ҳлҠ” л©”м„ңл“ңмқҙлӢӨ. BootstrapContext лһҖ м• н”ҢлҰ¬мјҖмқҙм…ҳ м»Ён…ҚмҠӨнҠёк°Җ мӨҖл№„лҗ  л•Ңк№Ңм§Җ нҷҳкІҪ ліҖмҲҳлҘј кҙҖлҰ¬н•ҳлҠ” Springмқҳ Environment к°қмІҙлҘј нӣ„мІҳлҰ¬н•ҳкё° мң„н•ң мһ„мӢң м»Ён…ҚмҠӨнҠёмқҙлӢӨ.

Spring BootлҠ” лӢӨм–‘н•ң нҷҳкІҪм—җм„ң м• н”ҢлҰ¬мјҖмқҙм…ҳмқ„ мӢӨн–үн•  мҲҳ мһҲмңјлҜҖлЎң, createBootstrapContext() лҠ” м• н”ҢлҰ¬мјҖмқҙм…ҳмқ„ мӢӨн–үн•  нҷҳкІҪкіј кҙҖл Ёлҗң BootstrapContextлҘј мғқм„ұн•ңлӢӨ. 

private DefaultBootstrapContext createBootstrapContext() {  
   DefaultBootstrapContext bootstrapContext = new DefaultBootstrapContext();  
   this.bootstrapRegistryInitializers.forEach((initializer) -> initializer.initialize(bootstrapContext));  
   return bootstrapContext;  
}

initializers лҰ¬мҠӨнҠёлҘј мҙҲкё°нҷ”н•ҳкі , н•ҙлӢ№ лҰ¬мҠӨнҠёмқҳ мҡ”мҶҢл“Өм—җ лҢҖн•ҙ initialize() л©”м„ңл“ңлҘј нҳём¶ңн•ҳм—¬ BootstrapContextлҘј мҙҲкё°нҷ”н•ңлӢӨ.

Java AWT Headless Property м„Өм •

Java AWT Headless PropertyлһҖ л””мҠӨн”Ңл Ҳмқҙ мһҘм№ҳк°Җ м—ҶлҠ” м„ңлІ„ нҷҳкІҪм—җм„ң UI нҒҙлһҳмҠӨлҘј мӮ¬мҡ©н•  мҲҳ мһҲлҸ„лЎқ н•ҳлҠ” мҳөм…ҳмқҙлӢӨ. java.awt.headless лҘј trueлЎң м„Өм •н•ҳкұ°лӮҳ, м„ңлІ„ мӢңмһ‘ мҠӨнҒ¬лҰҪнҠёмқҳ JAVA_OPTS нҷҳкІҪ ліҖмҲҳм—җ -D java.awt.headless=true лҘј м„Өм •н•ҳлҠ” л°©лІ• л“ұмқҙ мһҲлӢӨ.

private void configureHeadlessProperty() {  
   System.setProperty(SYSTEM_PROPERTY_JAVA_AWT_HEADLESS,  
         System.getProperty(SYSTEM_PROPERTY_JAVA_AWT_HEADLESS, Boolean.toString(this.headless)));  
}

мқҙ л¶Җл¶„мқҖ (мһҳ лӘЁлҘҙкІ мңјлӢҲ) мһҗм„ёнһҲ лӢӨлЈЁм§Җ м•Ҡкі  л„ҳм–ҙк°ҖкІ лӢӨ.

м• н”ҢлҰ¬мјҖмқҙм…ҳ м»Ён…ҚмҠӨнҠё мғқм„ұ

createApplicationContext лҠ” лӮҙл¶Җм ҒмңјлЎң ApplicationContextInitializer к°қмІҙл“Өмқҳ initialize() л©”м„ңл“ңлҘј нҳём¶ңн•ҳм—¬ ApplicationContext мқҳ мҙҲкё°нҷ” мһ‘м—…мқ„ мҲҳн–үн•ңлӢӨ. 

protected ConfigurableApplicationContext createApplicationContext() {  
   return this.applicationContextFactory.create(this.webApplicationType);  
}

мҪ”л“ңлҘј ліҙл©ҙ ApplicationContextFactory нҢ©нҶ лҰ¬ нҒҙлһҳмҠӨм—җкІҢ мғқм„ұмқ„ мң„мһ„н•ҳлҠ” кІғмқ„ ліј мҲҳ мһҲлӢӨ. м• н”ҢлҰ¬мјҖмқҙм…ҳ м»Ён…ҚмҠӨнҠё мғқм„ұ мӢңм—җлҠ” webApplicationType мқҙ мӮ¬мҡ©лҗңлӢӨ.

мҡ°лҰ¬мқҳ м• н”ҢлҰ¬мјҖмқҙм…ҳмқҖ SERVLET кё°л°ҳмңјлЎң мӢӨн–үлҗҳкё° л•Ңл¬ём—җ, нҷҳкІҪ лҳҗн•ң ApplicationServletEnvironment лЎң м„Өм •лҗҳм–ҙ мһҲлӢӨ.

лӮҙл¶ҖлҠ” лӢӨмқҢкіј к°ҷлӢӨ.

л””лІ„к№…мқ„ н•ҙліҙл©ҙ create()лЎң л¶Җн„° мғқм„ұлҗҳлҠ” ConfigurableApplicationContextлҠ” AnnotationConfigServletWebServerApplicationContextмқё кІғмқ„ нҷ•мқён•  мҲҳ мһҲлӢӨ.

Springмқҙ м ңкіөн•ҳлҠ” ApplicationContext кө¬нҳ„ нҒҙлһҳмҠӨм—җлҠ” м—¬лҹ¬ к°Җм§Җ мў…лҘҳк°Җ мһҲлӢӨ. Spring ApplicationмқҖ мӮ¬мҡ©мһҗ лҢҖмӢ  ApplicationContext мқҳ м Ғм Ҳн•ң нғҖмһ…мқ„ мғқм„ұн•ҳкІҢ лҗңлӢӨ.

WebApplicationType мқҖ лӢӨмқҢкіј к°ҷмқҙ м •н•ҙм§„лӢӨ.

  • Spring MVCк°Җ мЎҙмһ¬ -> AnnotaionConfigServletWebServerApplicationContext мӮ¬мҡ©
  • Spring MVCк°Җ мЎҙмһ¬н•ҳм§Җ м•Ҡкі , Spring WebFluxк°Җ мЎҙмһ¬н•ҳл©ҙ AnnotationConfigReactiveWebServerApplicationContext мӮ¬мҡ©
  • к·ё мҷём—җлҠ” AnnotationConfigApplicationContext мӮ¬мҡ©

мһ к№җ, AnnotationConfigServletWebServerApplicationContextлҠ” лӯҳк№Ң

ServletWebServerApplicationContext that accepts annotated classes as input - in particular @Configuration -annotated classes, but also plain @Component classes and JSR-330 compliant classes using javax.inject annotations. Allows for registering classes one by one (specifying class names as config location) as well as for classpath scanning (specifying base packages as config location).

Note: In case of multiple @Configuration classes, later @Bean definitions will override ones defined in earlier loaded files. This can be leveraged to deliberately override certain bean definitions through an extra Configuration class.

AnnotationConfigServletWebServerApplicationContext лҠ” мҡ°лҰ¬к°Җ Spring м• н”ҢлҰ¬мјҖмқҙм…ҳмқ„ мӮ¬мҡ©н•  л•Ң мӢӨн–үлҗҳлҠ” мӣ№ ApplicationContext мӨ‘мқҳ н•ҳлӮҳмқҙлӢӨ. Java ConfigлҘј мӮ¬мҡ©н•ҳм—¬ мӣ№ м• н”ҢлҰ¬мјҖмқҙм…ҳмқ„ м„Өм •н•ҳкё° мң„н•ҙ мӮ¬мҡ©лҗңлӢӨ.

мһ…л ҘмңјлЎң @Configuration м–ҙл…ён…Ңмқҙм…ҳ, @Component м–ҙл…ён…Ңмқҙм…ҳ, javax.inject м–ҙл…ён…Ңмқҙм…ҳмқ„ мӮ¬мҡ©н•ҳлҠ” JSR-330 compilant нҒҙлһҳмҠӨл“Өмқ„ л°ӣлҠ”лӢӨ. нҒҙлһҳмҠӨ мқҙлҰ„мқ„ м„Өм • мң„м№ҳлЎң нҠ№м •н•ҳм—¬ нҒҙлһҳмҠӨл“Өмқ„ л“ұлЎқн•ҳкі , base нҢЁнӮӨм§Җл“Өмқ„ м„Өм • мң„м№ҳлЎң нҠ№м •н•ҳм—¬ classpath scanningмқ„ н•ңлӢӨ.

@Configuration нҒҙлһҳмҠӨл“Өмқҙ м—¬лҹ¬ к°ң мһҲлҠ” кІҪмҡ°, мқҙнӣ„мқҳ @Bean м •мқҳл“ӨмқҖ лҚ” лЁјм Җ лЎңл“ңлҗң нҢҢмқјм—җм„ң м •мқҳлҗң beanмқ„ мҳӨлІ„лқјмқҙл”© н•ңлӢӨ. мқҙлҠ” нҠ№м • bean м •мқҳлҘј м¶”к°Җ configuration нҒҙлһҳмҠӨлҘј нҶөн•ҙ мқјл¶Җлҹ¬ мҳӨлІ„лқјмқҙл”©н•ҳкё° мң„н•ҙ мӮ¬мҡ©лҗ  мҲҳ мһҲлӢӨ.

private void prepareContext(DefaultBootstrapContext bootstrapContext, ConfigurableApplicationContext context,  
      ConfigurableEnvironment environment, SpringApplicationRunListeners listeners,  
      ApplicationArguments applicationArguments, Banner printedBanner) {  
    
   // EnvironmentлҘј м• н”ҢлҰ¬мјҖмқҙм…ҳ м»Ён…ҚмҠӨнҠём—җ м„Өм •н•ңлӢӨ. 
   context.setEnvironment(environment);  
   postProcessApplicationContext(context);  


   // Initializer initialize н•ҙмЈјкё°
   applyInitializers(context);  
   listeners.contextPrepared(context);  
   
   // BootstrapContext мў…лЈҢ
   bootstrapContext.close(context);  
   if (this.logStartupInfo) {  
      logStartupInfo(context.getParent() == null);  
      logStartupProfileInfo(context);  
   }  
   
   // Add boot specific singleton beans  
   ConfigurableListableBeanFactory beanFactory = context.getBeanFactory();  
   beanFactory.registerSingleton("springApplicationArguments", applicationArguments);  
   if (printedBanner != null) {  
      beanFactory.registerSingleton("springBootBanner", printedBanner);  
   }  
   if (beanFactory instanceof AbstractAutowireCapableBeanFactory) {  
      ((AbstractAutowireCapableBeanFactory) beanFactory).setAllowCircularReferences(this.allowCircularReferences);  
      if (beanFactory instanceof DefaultListableBeanFactory) {  
         ((DefaultListableBeanFactory) beanFactory)  
            .setAllowBeanDefinitionOverriding(this.allowBeanDefinitionOverriding);  
      }  
   }   if (this.lazyInitialization) {  
      context.addBeanFactoryPostProcessor(new LazyInitializationBeanFactoryPostProcessor());  
   }  
   context.addBeanFactoryPostProcessor(new PropertySourceOrderingBeanFactoryPostProcessor(context));  
   
   // Load the sources  
   Set<Object> sources = getAllSources();  
   Assert.notEmpty(sources, "Sources must not be empty");  
   load(context, sources.toArray(new Object[0]));  
   listeners.contextLoaded(context);  
}

@Override  
public void registerSingleton(String beanName, Object singletonObject) throws IllegalStateException {  
   Assert.notNull(beanName, "Bean name must not be null");  
   Assert.notNull(singletonObject, "Singleton object must not be null");  
   synchronized (this.singletonObjects) {  
      Object oldObject = this.singletonObjects.get(beanName);  
      if (oldObject != null) {  
         throw new IllegalStateException("Could not register object [" + singletonObject +  
               "] under bean name '" + beanName + "': there is already object [" + oldObject + "] bound");  
      }  
      addSingleton(beanName, singletonObject);  
   }  
}

Context Refresh лӢЁкі„

мӢұкёҖнҶӨ л№ҲмңјлЎң л“ұлЎқн•  нҒҙлһҳмҠӨл“Өмқ„ м°ҫм•„м„ң мғқм„ұн•ҳкі , нӣ„мІҳлҰ¬н•ҳлҠ” лӢЁкі„мқҙлӢӨ. refresh лҘј кұ°м№ҳл©ҙ лӘЁл“  к°қмІҙл“Өмқҙ мӢұкёҖнҶӨмңјлЎң мқёмҠӨн„ҙмҠӨнҷ” лҗҳлҠ”лҚ°, м—җлҹ¬к°Җ л°ңмғқн•ҳл©ҙ лӘЁл“  beanмқ„ м ңкұ°н•ңлӢӨ.

мҰү, refresh() лҘј кұ°міӨмқ„ л•Ң мӢұкёҖнҶӨмңјлЎң лӘЁл‘җ мқёмҠӨн„ҙмҠӨнҷ” лҗҳлҚҳм§Җ, м•„мҳҲ мқёмҠӨн„ҙмҠӨнҷ” лҗҳм§Җ м•Ҡм•„м•ј н•ңлӢӨ.

@Override  
public void refresh() throws BeansException, IllegalStateException {  
   synchronized (this.startupShutdownMonitor) {  
      StartupStep contextRefresh = this.applicationStartup.start("spring.context.refresh");  
  
      // Prepare this context for refreshing.  
      // RefreshлҘј мң„н•ҙ context мӨҖл№„н•ҳкё° 
      prepareRefresh();  
  
      // Tell the subclass to refresh the internal bean factory.  
      // лӮҙл¶Җ Bean Factory мӨҖл№„н•ҳкё°
      ConfigurableListableBeanFactory beanFactory = obtainFreshBeanFactory();  
  
      // Prepare the bean factory for use in this context.  
      // н•ҙлӢ№ contextм—җм„ң мӮ¬мҡ©н•ҳкё° мң„н•ң Bean Factory мӨҖл№„н•ҳкё°
      prepareBeanFactory(beanFactory);  
  
      try {  
         // Allows post-processing of the bean factory in context subclasses.  
         // Bean Factoryмқҳ нӣ„мІҳлҰ¬ м§„н–ү
         postProcessBeanFactory(beanFactory);  
  
         StartupStep beanPostProcess = this.applicationStartup.start("spring.context.beans.post-process");  
         
         // Invoke factory processors registered as beans in the context.  
         // Contextм—җ beanмңјлЎң л“ұлЎқлҗҳлҠ” Factory processor мӢӨн–ү (BeanFactoryPostProcessor мӢӨн–ү)
         invokeBeanFactoryPostProcessors(beanFactory);  
  
         // Register bean processors that intercept bean creation.  
         // Bean мғқм„ұмқ„ к°ҖлЎңмұ„лҠ” Bean Processor л“ұлЎқ (BeanPostProcessor л“ұлЎқ)
         registerBeanPostProcessors(beanFactory);  
         beanPostProcess.end();  
  
         // Initialize message source for this context.  
         // messageSource л°Ҹ Event Multicaster мҙҲкё°нҷ”
         initMessageSource();  
  
         // Initialize event multicaster for this context.  
         initApplicationEventMulticaster();  
  
         // Initialize other special beans in specific context subclasses.  
         // мӣ№ м„ңлІ„ мғқм„ұ
         onRefresh();  
  
         // Check for listener beans and register them.  
         // listener л№Ҳл“Өмқ„ мЎ°нҡҢ л°Ҹ л“ұлЎқ
         registerListeners();  
  
         // Instantiate all remaining (non-lazy-init) singletons.  
         // л№Ҳ мқёмҠӨн„ҙмҠӨнҷ” л°Ҹ нӣ„мІҳлҰ¬
         finishBeanFactoryInitialization(beanFactory);  
  
         // Last step: publish corresponding event.  
         finishRefresh();  
      }  
  
      catch (BeansException ex) {  
         if (logger.isWarnEnabled()) {  
            logger.warn("Exception encountered during context initialization - " +  
                  "cancelling refresh attempt: " + ex);  
         }  
  
         // Destroy already created singletons to avoid dangling resources.  
         destroyBeans();  
  
         // Reset 'active' flag.  
         cancelRefresh(ex);  
  
         // Propagate exception to caller.  
         throw ex;  
      }  
  
      finally {  
         // Reset common introspection caches in Spring's core, since we  
         // might not ever need metadata for singleton beans anymore...         resetCommonCaches();  
         contextRefresh.end();  
      }  
   }}

prepareBeanFactory() м—җм„ң мқҳмЎҙм„ұ мЈјмһ…мқ„ л¬ҙмӢңн•  мқён„°нҺҳмқҙмҠӨл“Өмқ„ л“ұлЎқн•ңлӢӨ.

protected void prepareBeanFactory(ConfigurableListableBeanFactory beanFactory) {  
   // Tell the internal bean factory to use the context's class loader etc.  
   beanFactory.setBeanClassLoader(getClassLoader());  
   if (!shouldIgnoreSpel) {  
      beanFactory.setBeanExpressionResolver(new StandardBeanExpressionResolver(beanFactory.getBeanClassLoader()));  
   }  
   beanFactory.addPropertyEditorRegistrar(new ResourceEditorRegistrar(this, getEnvironment()));  
  
   // Configure the bean factory with context callbacks.  
   beanFactory.addBeanPostProcessor(new ApplicationContextAwareProcessor(this));  
   beanFactory.ignoreDependencyInterface(EnvironmentAware.class);  
   beanFactory.ignoreDependencyInterface(EmbeddedValueResolverAware.class);  
   beanFactory.ignoreDependencyInterface(ResourceLoaderAware.class);  
   beanFactory.ignoreDependencyInterface(ApplicationEventPublisherAware.class);  
   beanFactory.ignoreDependencyInterface(MessageSourceAware.class);  
   beanFactory.ignoreDependencyInterface(ApplicationContextAware.class);  
   beanFactory.ignoreDependencyInterface(ApplicationStartupAware.class);  
  
   // BeanFactory interface not registered as resolvable type in a plain factory.  
   // MessageSource registered (and found for autowiring) as a bean.   beanFactory.registerResolvableDependency(BeanFactory.class, beanFactory);  
   beanFactory.registerResolvableDependency(ResourceLoader.class, this);  
   beanFactory.registerResolvableDependency(ApplicationEventPublisher.class, this);  
   beanFactory.registerResolvableDependency(ApplicationContext.class, this);  
  
   // Register early post-processor for detecting inner beans as ApplicationListeners.  
   beanFactory.addBeanPostProcessor(new ApplicationListenerDetector(this));  
  
   // Detect a LoadTimeWeaver and prepare for weaving, if found.  
   if (!NativeDetector.inNativeImage() && beanFactory.containsBean(LOAD_TIME_WEAVER_BEAN_NAME)) {  
      beanFactory.addBeanPostProcessor(new LoadTimeWeaverAwareProcessor(beanFactory));  
      // Set a temporary ClassLoader for type matching.  
      beanFactory.setTempClassLoader(new ContextTypeMatchClassLoader(beanFactory.getBeanClassLoader()));  
   }  
  
   // Register default environment beans.  
   if (!beanFactory.containsLocalBean(ENVIRONMENT_BEAN_NAME)) {  
      beanFactory.registerSingleton(ENVIRONMENT_BEAN_NAME, getEnvironment());  
   }  
   if (!beanFactory.containsLocalBean(SYSTEM_PROPERTIES_BEAN_NAME)) {  
      beanFactory.registerSingleton(SYSTEM_PROPERTIES_BEAN_NAME, getEnvironment().getSystemProperties());  
   }  
   if (!beanFactory.containsLocalBean(SYSTEM_ENVIRONMENT_BEAN_NAME)) {  
      beanFactory.registerSingleton(SYSTEM_ENVIRONMENT_BEAN_NAME, getEnvironment().getSystemEnvironment());  
   }  
   if (!beanFactory.containsLocalBean(APPLICATION_STARTUP_BEAN_NAME)) {  
      beanFactory.registerSingleton(APPLICATION_STARTUP_BEAN_NAME, getApplicationStartup());  
   }  
}
