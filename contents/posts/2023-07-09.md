## 배운 것

- dp로 LIS 풀기

```python
dp = [1 for _ in range(n)]  
for i in range(n):  
    for j in range(i):  
        if arr[i] > arr[j]:  
            dp[i] = max(dp[i], dp[j] + 1)
```

i번째보다 작은 값들을 돌면서 현재 i 값보다 작은 경우 현재 dp 값, 이전 dp값 + 1 중 최댓값을 확인한 후 업데이트한다.
