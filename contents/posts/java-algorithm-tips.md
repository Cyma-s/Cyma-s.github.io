## 자료구조

- `Queue` -> `PriorityQueue`
	- `add`: 데이터 삽입
	- `peek`: 맨 앞 값 반환
	- `poll`: 맨 앞 값 반환 후 삭제
- `Deque` -> `LinkedList`

## 유용한 메서드

- array to List
	- `Arrays.asList(배열)`
- List to array
	- `리스트.toArray(할당할 배열)`
