WebMvcTest м–ҙл…ён…Ңмқҙм…ҳ

WebMvcTest лҠ” Spring MVC controllerлҘј н…ҢмҠӨнҠён•ҳкё° мң„н•ҙ мӮ¬мҡ©лҗҳлҠ” Spring Bootмқҳ м–ҙл…ён…Ңмқҙм…ҳмқҙлӢӨ. мқҙ м–ҙл…ён…Ңмқҙм…ҳмқҖ м „мІҙ auto-configurationмқ„ л№„нҷңм„ұнҷ”н•ҳкі  лҢҖмӢ  MVC н…ҢмҠӨнҠёмҷҖ кҙҖл Ёлҗң кө¬м„ұл§Ң м Ғмҡ©н•ңлӢӨ. мҰү, Application Context лҘј мҷ„м „н•ҳкІҢ Start н•ҳм§Җ м•Ҡкі  Web LayerлҘј н…ҢмҠӨнҠё н•ҳкі  мӢ¶мқ„ л•Ң @WebMvcTest лҘј мӮ¬мҡ©н•ҳлҠ” кІғмқ„ кі л Өн•ҙліј мҲҳ мһҲлӢӨ.

лӢЁмң„ н…ҢмҠӨнҠёлҘј мң„н•ҙ Spring MVC мқён”„лқјлҘј мһҗлҸҷмңјлЎң кө¬м„ұн•ҳм§Җл§Ң, мҠӨмә”лҗҳлҠ” Beanмқ„ @Controller, @ControllerAdvice, @RestController, @JsonComponent, Filter, WebMvcConfigurer, HandlerMethodArgumentResolver лЎң м ңн•ңн•ңлӢӨ.
л§Ңм•Ҫ ServiceлӮҳ Repositoryмқҳ Dependencyк°Җ н•„мҡ”н•ң кІҪмҡ°м—җлҠ” @MockBean мңјлЎң мЈјмһ… л°ӣм•„ н…ҢмҠӨнҠёлҘј м§„н–үн•ңлӢӨ.

@WebMvcTestлҘј мӮ¬мҡ©н•  л•Ң, мқјл°ҳ @Component, @Service, @Repository beanмқҖ мҠӨмә”лҗҳм§Җ м•ҠлҠ”лӢӨ.
л”°лқјм„ң @MockBean лҳҗлҠ” @SpyBean мқ„ мӮ¬мҡ©н•ҳм—¬ к°Җм§ң к°қмІҙлҘј beanмңјлЎң л“ұлЎқн•ҙмЈјм–ҙм•ј н•ңлӢӨ.

@SpringBootTest

н”„лЎңм қнҠё м•Ҳмқҳ лӘЁл“  beanмқ„ л“ұлЎқн•ҳм—¬ н…ҢмҠӨнҠёлҘј н•ңлӢӨ. лӢЁмң„ н…ҢмҠӨнҠёмІҳлҹј кё°лҠҘмқ„ н…ҢмҠӨнҠён•  л•ҢліҙлӢӨлҠ” нҶөн•© н…ҢмҠӨнҠёлҘј н•  л•Ң мӮ¬мҡ©н•ңлӢӨ.

м„ңлІ„лҘј лқ„мҡ°кі  лӘЁл“  beanмқ„ л“ұлЎқн•ҳкё° л•Ңл¬ём—җ лӢӨм–‘н•ң н…ҢмҠӨнҠё мӨ‘м—җм„ң к°ҖмһҘ мҡҙмҳҒнҷҳкІҪкіј мң мӮ¬н•ң н…ҢмҠӨнҠёмқҙлӢӨ.

к·ёлҹ¬лӮҳ лӘЁл“  beanмқ„ лЎңл“ңн•ҳкё° л•Ңл¬ём—җ н…ҢмҠӨнҠё кө¬лҸҷ мӢңк°„мқҙ мҳӨлһҳ кұёлҰ¬кі , н…ҢмҠӨнҠё лӢЁмң„к°Җ нҒ¬кё° л•Ңл¬ём—җ л””лІ„к№…мқҙ м–ҙл Өмҡё мҲҳ мһҲлӢӨ.

Controller л Ҳмқҙм–ҙл§Ң н…ҢмҠӨнҠён•ҳкі  мӢ¶мқ„ л•Ңм—җлҠ” @WebMvcTest лҘј м“°лҠ” кІғмқҙ мң мҡ©н•ҳлӢӨ.

WebMvcTest мӮ¬мҡ©н•ҳкё°

кё°ліём ҒмңјлЎң @WebMvcTest лЎң мЈјм„қмқ„ лӢЁ н…ҢмҠӨнҠёлҠ” Spring Security л°Ҹ MockMvcлҸ„ мһҗлҸҷмңјлЎң кө¬м„ұн•ңлӢӨ.

н…ҢмҠӨнҠё нҒҙлһҳмҠӨм—җ @WebMvcTest м–ҙл…ён…Ңмқҙм…ҳмқ„ лӢ¬кі  н…ҢмҠӨнҠён•  м»ЁнҠёлЎӨлҹ¬лҘј м§Җм •н•ңлӢӨ.

@WebMvcTest(RacingCarController.class)  
class RacingCarControllerTest

мқҙл ҮкІҢ н…ҢмҠӨнҠён•  м»ЁнҠёлЎӨлҹ¬лҘј м§Җм •н•ҳкІҢ лҗҳл©ҙ м „мІҙ м• н”ҢлҰ¬мјҖмқҙм…ҳ м»Ён…ҚмҠӨнҠёк°Җ м•„лӢҢ RacingCarController л§Ң кө¬м„ұн•ҳкІҢ лҗңлӢӨ.

н•ҙлӢ№ н…ҢмҠӨнҠём—җ м Ғмҡ©н•ҙм„ңлҠ” м•Ҳ лҗҳлҠ” auto-configuration нҒҙлһҳмҠӨлҘј м ңмҷён•ҳлҸ„лЎқ excludeAutoConfiguration мҳөм…ҳмқ„ м§Җм •н•  мҲҳлҸ„ мһҲлӢӨ.

@WebMvcTest(
    controllers = RacingCarController.class,
    excludeAutoConfiguration = SecurityAutoConfiguration.class,

excludeFilters : м¶”к°ҖлҗҳлҠ” beanл“Ө мӨ‘м—җ м ңмҷён•ҳкі  мӢ¶мқҖ Beanмқҳ н•„н„°лҘј л“ұлЎқн•ңлӢӨ. мҰү, classesм—җ л“Өм–ҙк°Ҳ Beanмқ„ м ңмҷён•ҳкё° мң„н•ң н•„н„°лҘј м •мқҳн•ҳлҠ” кІғмқҙлӢӨ. ASSIANABLE_TYPE мҶҚм„ұмқҖ м ңмҷён•  кё°мӨҖмқ„ нҒҙлһҳмҠӨлЎң м§Җм •н•ңлӢӨ.

лҳҗн•ң @MockBean мқ„ мӮ¬мҡ©н•ҙм„ң н•„мҡ”н•ң мқҳмЎҙм„ұм—җ лҢҖн•ҙ mock кө¬нҳ„мқ„ м ңкіөн•  мҲҳлҸ„ мһҲлӢӨ.

Web Layer кҙҖл Ё л№Ҳл“Өл§Ң л“ұлЎқн•ҳкё° л•Ңл¬ём—җ, м»ЁнҠёлЎӨлҹ¬лҠ” мЈјмһ…мқҙ м •мғҒм ҒмңјлЎң лҗҳм§Җл§Ң, @Component лЎң л“ұлЎқлҗң RepositoryмҷҖ ServiceлҠ” мЈјмһ…мқҙ лҗҳм§Җ м•ҠлҠ”лӢӨ.

л”°лқјм„ң, @WebMvcTest м—җм„ң RepositoryмҷҖ ServiceлҘј мӮ¬мҡ©н•ҳкё° мң„н•ҙм„ңлҠ” @MockBean мқ„ мӮ¬мҡ©н•ҳм—¬ beanмңјлЎң л“ұлЎқн•ҙмЈјм–ҙм•ј н•ңлӢӨ.

@MockBean мқҙлһҖ к°Җм§ң к°қмІҙлЎң, нҳём¶ңкіј кІ°кіјлҘј мһ„мқҳлЎң мЎ°мһ‘н•ҳм—¬ н•ҙлӢ№ лӢЁмң„ н…ҢмҠӨнҠём—җл§Ң м§‘мӨ‘н•  мҲҳ мһҲлҸ„лЎқ лҸ„мҷҖмӨҖлӢӨ.

н…ҢмҠӨнҠё мҪ”л“ң мһ‘м„ұн•ҙліҙкё°

@Test  
    void play_л©”м„ңл“ңк°Җ_м Ғм Ҳн•ң_нҳ•мӢқмқ„_л°ҳнҷҳн•ңлӢӨ() throws Exception {  
        final PlayRequest playRequest = new PlayRequest("лёҢлҰ¬,нҶ лҜё,лёҢлқјмҡҙ", 1);  
        final String request = objectMapper.writeValueAsString(playRequest);  
        Car bri = new Car("лёҢлҰ¬");  
        Car tomi = new Car("нҶ лҜё");  
        Car brown = new Car("лёҢлқјмҡҙ");  
        List<Car> cars = List.of(bri);  
        final PlayResponse playResponse = PlayResponse.of(  
                RacingCarWinnerDto.of(cars),  
                List.of(  
                        RacingCarStatusDto.from(bri),  
                        RacingCarStatusDto.from(tomi),  
                        RacingCarStatusDto.from(brown)

        given(randomMoveStrategy.isMovable()).willReturn(true);

                .willReturn(playResponse);

                        .content(request)  
                        .contentType(MediaType.APPLICATION_JSON))  
                .andExpect(status().isOk())  
                .andExpect(jsonPath("$.winners[0]").value("лёҢлҰ¬"))  
                .andExpect(jsonPath("racingCars", hasSize(3)))  
                .andExpect(jsonPath("$.racingCars[0].name").value("лёҢлҰ¬"))  
                .andExpect(jsonPath("$.racingCars[0].position").value(0));  
    }
}

given(), when(), willReturn(), thenReturn() л©”м„ңл“ңл“ӨмқҖ BDDMockitoм—җ м •мқҳлҗҳм–ҙ мһҲлӢӨ.

@MockBean мңјлЎң л“ұлЎқн•ң Serviceмқҳ н–үлҸҷмқ„ given() мңјлЎң м§Җм •н•ҙмӨҖлӢӨ.
willReturn() м—җм„ңлҠ” н•ҙлӢ№ н–үлҸҷм—җм„ң м–ҙл–Ө к°’мқ„ лҰ¬н„ҙн•ҙм•ј н•ҳлҠ”м§Җ м§Җм •н•ңлӢӨ.

perform() м—җм„ң м»ЁнҠёлЎӨлҹ¬м—җ мҡ”мІӯмқ„ м „мҶЎн•ңлӢӨ.
andExpect()м—җм„ңлҠ” мқ‘лӢөмқ„ кІҖмҰқн•ҳлҠ” м—ӯн• мқ„ н•ңлӢӨ.
status() мқҖ мғҒнғң мҪ”л“ңлҘј кІҖмҰқн•ңлӢӨ. content() мқҖ мқ‘лӢөм—җ лҢҖн•ң м •ліҙлҘј кІҖмҰқн•  мҲҳ мһҲлӢӨ.
